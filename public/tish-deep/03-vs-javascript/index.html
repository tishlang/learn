Source of truth: LANGUAGE.md ("Omitted vs typical JS" section).

Removed

  • undefined ‚Äî Tish has only null.
  • == / != ‚Äî strict ===/!== only.
  • var ‚Äî let/const only; block-scoped, no hoisting.
  • this ‚Äî pass values explicitly.
  • class / super ‚Äî plain objects + closures instead.
  • prototypes / instanceof
  • for..in
  • generators, Symbol, BigInt, Map, Set
  • delete

Changed

  • typeof null is "null" (JS gives "object").
  • ?. yields null instead of undefined.
  • String indices follow Unicode scalar values (Rust char), matching BMP JS strings. Astral characters (some emoji) differ ‚Äî one Tish index, two UTF-16 code units in JS.
  • Bitwise ops use 32-bit integer semantics.
