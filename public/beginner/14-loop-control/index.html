Capstone: FizzBuzz

The classic interview question. For numbers 1 through 15:

  • If divisible by 3 and 5: print "FizzBuzz".
  • Else if divisible by 3: print "Fizz".
  • Else if divisible by 5: print "Buzz".
  • Else: print the number.
