Track 3 — Capstones

If you want to build complete projects: real-time chat, Snake the polished version, a static blog generator, a REST API, a collaborative whiteboard, your own tiny game engine — pick a capstone.

Tooling and editor

  • REPL — tish with no args drops you into an interactive prompt. Try expressions, see results.
  • VS Code — install the Tish extension for syntax highlighting, formatting, and Go-to-definition.
  • Tish-fmt — tish-fmt yourfile.tish formats your code consistently.

The wider world

  • tishlang.com — full reference docs.
  • LANGUAGE.md — the canonical spec.
  • The tish-playground workspace — paste any Tish code, run it instantly. Same editor + runner this lesson site uses.

A parting thought

The most important thing isn't memorizing every API. It's the habit you've built: write a tiny piece, run it, see it, repeat. That habit will carry you through every language, framework, and library you ever touch.

Keep going.
